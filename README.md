# MyChickenBreastShop

## 프로젝트 개요
닭가슴살 쇼핑몰 API 입니다.  
상품 조회, 검색 기능이 있습니다.  
Kakao REST API를 이용해 테스트 구매와 주문까지 가능하며,  
네이버 SMTP를 이용해 회원가입 유저는 이메일로 인증번호를 전송 받습니다.

## 프로젝트 기간
- 2022.04.28 ~ 

## 사용 기술 스택
```
Java 11, Gradle
Spring Boot
Mybatis
Mysql 8.0.17
```
